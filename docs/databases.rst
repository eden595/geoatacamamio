Database
========

Para este proyecto utilizamos una DB relacional de ``MySql``, esta base de datos gestiona la información relacionada con la administración de usuarios, permisos, checklist de materiales, perforaciones, maquinaria y empresas. Su propósito es permitir el control eficiente de operaciones en el ámbito industrial, específicamente en la minería y perforación.

Resumen de la Base de Datos
---------------------------

1. **Usuarios y Permisos**

    Módulo encargado de la gestión de autenticación y autorización, permitiendo la administración de roles y permisos dentro del sistema.

1. **Gestión de Checklists**

    Incluye tablas para el registro y seguimiento de materiales utilizados en casetas y sondas, asegurando el control de documentación y estado de los elementos requeridos en el proceso operativo.

2. **Empresas y Mantenimiento**

    Contiene información sobre las empresas registradas en el sistema, así como los servicios de mantenimiento asociados a cada una de ellas.

3. **Perforaciones y Maquinaria**

    Este módulo almacena datos sobre las perforaciones realizadas, las herramientas utilizadas y la maquinaria involucrada, facilitando la trazabilidad y control de las operaciones.

4. **Reportes Operacionales**

    Gestiona reportes digitales relacionados con el consumo de insumos, tiempos de trabajo y progresos de perforación, proporcionando una visión detallada del rendimiento en terreno.

Relaciones entre Tablas
-----------------------

Las tablas están interconectadas mediante claves foráneas para garantizar la integridad referencial de los datos. 
Se han definido relaciones clave entre usuarios, checklist, perforaciones y maquinaria para mantener la coherencia de la información.
